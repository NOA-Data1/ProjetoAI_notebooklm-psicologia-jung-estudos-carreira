# Psicologia Junguiana para Estudos e Carreira  
### Projeto com IA usando NotebookLM


## Contexto e Objetivos

Este projeto tem como objetivo utilizar Inteligência Artificial, por meio do NotebookLM, para criar um "segundo cérebro" baseado na psicologia de Carl Jung.

A proposta é aplicar conceitos profundos da psicologia analítica para resolver desafios modernos como:

- Falta de foco nos estudos (névoa mental)
- Ansiedade em relação ao futuro
- Transição de carreira profissional

O projeto une tecnologia + autoconhecimento + estratégia profissional.

## 💬 Engenharia de Prompts (Cicatrizes)

### Prompt refinado:

### Aprendizados (troubleshooting):

- Prompts genéricos geram respostas fracas  
- Contexto e direcionamento mudam completamente a qualidade  
- Pedir "como Jung" aumenta profundidade  
- Iteração é essencial (testar → ajustar → melhorar)

## Miniguia de Estudo

### Resumo

A falta de foco e a ansiedade não são apenas problemas externos, mas sinais de conflitos internos.

Segundo Carl Jung:

- A **Sombra** representa partes reprimidas que consomem energia  
- A **Persona** pode causar exaustão quando usada em excesso  
- A **Energia Psíquica (libido)** precisa estar alinhada com propósito  
- A **Individuação** é o caminho para clareza e realização  


### 📖 Glossário

- **Persona:** máscara social que usamos no trabalho e na vida  
- **Sombra:** aspectos reprimidos da personalidade  
- **Individuação:** processo de se tornar quem você realmente é  
- **Energia Psíquica:** motivação e foco mental  
- **Self:** totalidade do indivíduo
